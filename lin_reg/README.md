# Linear Regression Tutorial

This folder contains a paired Python and R workflow for simple linear regression on the `regression_data.csv` dataset. The project includes interactive Jupyter notebooks, exported HTML reports, and standalone command-line scripts.

## Dataset

`regression_data.csv` contains salary observations paired with years of experience:

| Column | Description |
|--------|-------------|
| `YearsExperience` | Independent variable (years) |
| `Salary` | Dependent variable (annual salary) |

## Purpose of Assignment

Build equivalent linear regression applications in Python and R that:

1. Read a CSV file
2. Create a scatter plot of the chosen variables
3. Fit a linear regression model
4. Overlay the regression line on the plot
5. Evaluate model performance (R-squared and related metrics)
6. Export results as notebooks, HTML, and CLI-driven plot images

## Files

| File | Description |
|------|-------------|
| `regression_data.csv` | Input dataset |

## Tools and Libraries

**Python**

- `pandas` — data loading
- `matplotlib` — plotting
- `scikit-learn` — `LinearRegression`, `r2_score`, `mean_squared_error`

**R**

- Base `stats` — `lm()`, `summary()`
- `ggplot2` — scatter plot with regression line overlay

**Environment**

- Conda environment: `7030_class_1` (see `environment.yml` and `setup_env.sh`)

## Running the Notebooks

From the `lin_reg` directory:

```bash
jupyter lab python_lin_reg.ipynb
jupyter lab r_lin_reg.ipynb
```

Make sure the working directory is `lin_reg` so the notebooks can find `regression_data.csv`.

## Running the CLI Scripts

Both scripts accept three arguments: `<filename> <x_column> <y_column>`.

**Python**

```bash
python python_lin_reg.py regression_data.csv YearsExperience Salary
```

**R**

```bash
Rscript r_lin_reg.r regression_data.csv YearsExperience Salary
```

Each script prints model coefficients and evaluation metrics, then saves a plot:

- Python → `lin_reg_py_output.png`
- R → `lin_reg_r_output.png`

## Results

On the sample dataset (`YearsExperience` predicting `Salary`), both implementations produce consistent results:

- **R-squared:** ~0.785
- **Trend:** Salary increases with years of experience
- **Visualization:** Red points (observations) with a blue regression line overlay

The exported HTML notebooks and PNG outputs document the full analysis for review without re-running the code.

## Exporting HTML

To regenerate HTML exports from the notebooks:

```bash
jupyter nbconvert --to html python_lin_reg.ipynb
jupyter nbconvert --to html r_lin_reg.ipynb
```
