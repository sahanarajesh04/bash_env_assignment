Linear Regression in Python¶

This notebook reads regression_data.csv, visualizes the relationship between years of experience and salary, fits a simple linear regression model, overlays the fitted line on the scatter plot, and evaluates model performance.

import pandas as pd
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

X_COLUMN = "YearsExperience"
Y_COLUMN = "Salary"

dataset = pd.read_csv("regression_data.csv")

model = LinearRegression()
X = dataset[[X_COLUMN]]
y = dataset[Y_COLUMN]
model.fit(X, y)

print(f"Intercept: {model.intercept_:.2f}")
print(f"Coefficient ({X_COLUMN}): {model.coef_[0]:.2f}")
